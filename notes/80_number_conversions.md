
# Number Conversions


## Binary to Hex and back

 - Hex: 1, 2, 3, 4, 5, 6, 7, 8, 9, A, B, C, D, E, F
 - Binary: 1010 = A, 1001(9), etc.

 Hex   |   Binary
  A    ->   1010
  9    ->   1001 
  3    <-   0011
  5    <-   0101


## Decimal to Hex an back

Decimal: 16^3, 16^2, 16^1, 16^0

  Hex   |   Decimal
0x7B93  ->   31,635
0x0BC9  <-   3,017  Truncate the number
